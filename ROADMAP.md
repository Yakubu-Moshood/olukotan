# Roadmap

## Delivered in Phase 0–1 source

- Tauri/React/TypeScript/Rust foundation and SQLite index
- branded home, chosen-folder project creation, manifest and portable file tree
- recent projects, pin/remove/reveal, open and schema validation
- basic UTF-8 Fountain editor, scene navigator and approximate counts
- explicit save, idle autosave, readable crash recovery, read-only mode and external-change guard

## Phase 2 — screenplay editor

Build a structured element model and keyboard state machine; professional A4/US Letter pagination; scene/character autocomplete; title page; dual dialogue; stable PDF export; find/replace; focus/full-screen modes; and richer recovery/conflict UI. Test cursor behaviour and long-script performance before virtualisation.

## Phase 3 — compatibility

Add guided Fountain/FDX import and export first, with previews, retained originals, warnings, fixtures, and round-trip tests. Then add general document formats in the agreed priority order.

## Later phases

Story development, document library, decisions, versions, production revisions and reports follow only after the editor/import foundation is reliable. Optional AI remains last, separate, explicit, and non-destructive.
